# Blinkit-Sales-Analysis
This project analyzes Blinkit sales data using SQL, Python, and Power BI.

Tools Used:
- SQL
- Python (Jupyter Notebook)
- Power BI
- Excel

Project Files:
- BlinkIT.sql → SQL queries for analysis
- Blinkit_Analysis.ipynb → Python data analysis
- Blinkit Dashboard.pbix → Power BI dashboard
